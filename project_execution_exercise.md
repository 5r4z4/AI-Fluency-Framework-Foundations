# Exercise: Project Execution with Description-Discernment Loops 
> এই নোটটা [Discernment Deep Dive নোট](discernment_deep_dive_bangla_eng.md), [Delegation Exercise নোট](delegation_exercise_bangla_eng.md) এবং বাকি সিরিজের সাথে link করা। এখানে তুমি Lesson 5-এ পরিকল্পনা করা project-টা **Description ও Discernment** skill ব্যবহার করে বাস্তবে execute করবে।

Course: *AI Fluency: Framework & Foundations*

---

## Step 1: তোমার Project Plan Review করো

- Lesson 5-এ তৈরি করা project plan-টা বের করো
- তোমার delegation decision গুলো দ্রুত review করো — কোন task human expertise-এ ভালো হবে, কোনটা AI capability-তে, কোনটা collaboration-এ
- তখন থেকে যা শিখেছো তার ভিত্তিতে চাইলে plan-টা refine করতে পারো

---

## Step 2: তোমার Description Approach তৈরি করো

Claude-এর সাথে conversation শুরু করো এবং কোন project নিয়ে একসাথে কাজ করবে তা ব্যাখ্যা করো। Execution শুরু করার আগে, নিজের **Description** approach পরিকল্পনা করো:

| Description Type | নিজেকে জিজ্ঞাসা করো |
|---|---|
| **Product Description** | প্রতিটা task-এর জন্য Claude থেকে কোন নির্দিষ্ট output চাই? Format, style, length, ও detail-এর level কেমন হবে? |
| **Process Description** | Claude প্রতিটা task কীভাবে approach করবে? নির্দিষ্ট কোনো method, framework, বা step follow করাতে চাও? |
| **Performance Description** | এই project-এ Claude-এর সাথে কেমন collaborative behavior চাও? Concise নাকি detailed, challenging নাকি supportive, idea-focused নাকি analysis-focused? |

এই প্রশ্নগুলো Claude-এর সাথে আলোচনা করে collaboration-এর জন্য স্পষ্ট প্রত্যাশা (expectation) ঠিক করো।

---

## Step 3: Description-Discernment Loop দিয়ে Project Execute করো

তোমার planned task গুলো Claude-এর সাথে একে একে করো। প্রতিটা task-এর জন্য এই loop follow করো:

### 1️⃣ Describe (স্পষ্টভাবে বলো)
তুমি যা শিখেছো সেই Description skill ব্যবহার করে:
- ঠিক কী চাও তা specific করে বলো (**Product**)
- Claude task-টা কীভাবে approach/চিন্তা করবে তা guide করো (**Process**)
- Process চলাকালীন Claude কীভাবে তোমার সাথে engage করবে তা বলো (**Performance**)

### 2️⃣ Discern (যাচাই করো)
যা পেলে সেটার quality যাচাই করো:
- Output-টা নিজে মূল্যায়ন করো (**Product Discernment**)
- Claude task-টা কীভাবে approach করেছে তা বিবেচনা করো (**Process Discernment**)
- Claude-এর behavior তোমার প্রয়োজনের জন্য সবচেয়ে সহায়ক কিনা বিবেচনা করো (**Performance Discernment**)

### 3️⃣ Refine (পরিমার্জন করো)
তোমার Discernment-এর ভিত্তিতে:
- কী কাজ করেছে আর কী করেনি তার feedback দাও
- প্রয়োজনমতো তোমার description clarify বা adjust করো
- সন্তুষ্ট না হওয়া পর্যন্ত iteration চাও

### 4️⃣ Integrate (নিজের Expertise যোগ করো)
- তোমার unique perspective, creativity, বা domain knowledge যোগ করো
- কী রাখবে, কী modify করবে, কী বাদ দেবে — চূড়ান্ত সিদ্ধান্ত তুমি নাও
- Final output-এর দায়িত্ব (responsibility) নিজে নাও

💡 প্রতিটা task-এর জন্য এই **Description → Discernment → Refine → Integrate** loop চালিয়ে যাও, যতক্ষণ না পুরো project সম্পূর্ণ হয়।

## 🪞 Reflection (নিজেকে জিজ্ঞাসা করো)

- কোন ধরনের description সবচেয়ে ভালো ফলাফল দিয়েছে — কোনো pattern লক্ষ্য করেছো?
- Description নাকি Discernment — কোনটাতে তোমার বেশি effort লেগেছে? কেন মনে হয় এমন হলো?
- তোমার আসল project execution, Lesson 5-এর initial plan-এর সাথে কেমন মিলল? পথে কী কী adjustment করতে হয়েছে?

## ➡️ পরবর্তী Lesson

পরের lesson-এ আমরা AI Fluency Framework-এর শেষ competency **Diligence** নিয়ে explore করব। Delegation, Description, ও Discernment মূলত effectiveness ও efficiency নিয়ে কাজ করে, কিন্তু **Diligence** AI-এর সাথে কাজ করার **ethical ও safety** দিকটা নিয়ে কাজ করে — কীভাবে তোমার AI collaboration দায়িত্বশীল (responsible), স্বচ্ছ (transparent), এবং accountable রাখবে তা শিখবে।
